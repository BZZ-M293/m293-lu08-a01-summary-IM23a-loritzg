Zusammenfassung Modul 293

  • body: text im browser
  • nav: Navigation im header
  • main: hauptteil
  • article: geschlossener teil in main
  • section: Absatz in article
  • p: paragraf im main
  • ul: liste mit punkten
  • ol: liste mit zahlen
  • a: link zu einer webseite
  • strong: fest schreiben
  • b: das gleiche wie strong
  • em: schräg schreiben
  • input: textfeld
  • br: zeilenumbruch
  • img: bilder einfügen
  • audio: audio einfügen
  • button: klickbarer knopf
  • datalist: input mit vorschlägen
  • div: möglichst nicht zunutzende section oder article
  • fieldset: zusammenhängende elemente gruppieren
  • h1-h6: grösse des textes
  • legend: titel für fieldset
  • li: item in einer liste
  • optgroup: titel für optionen in select
  • option: option in einem select
  • video: video einfügen
